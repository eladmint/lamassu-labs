Fix Applied

Problem: JavaScript was preventing default navigation behavior for partnership links

Solution: Modified setupNavigation() function to check if nav item has href but no data-section, then allow default navigation

Code Change: Added conditional logic to distinguish between internal sections and external navigation

How the Fix Works

// Old behavior: Always prevented default navigation
navItems.forEach(item => {
    item.addEventListener('click', (e) => {
        e.preventDefault(); // ❌ This blocked ALL navigation
        // ... internal section logic
    });
});

// New behavior: Smart navigation detection
navItems.forEach(item => {
    item.addEventListener('click', (e) => {
        const href = item.getAttribute('href');
        const section = item.dataset.section;

        if (href && href !== '#' && !section) {
            return; // ✅ Allow default navigation for partnership links
        }

        e.preventDefault(); // Only prevent for internal sections
        // ... internal section logic
    });
});
